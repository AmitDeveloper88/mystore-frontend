# mystore-frontend
Installation

Clone the Repository:

git clone https://github.com/AmitDeveloper88/mystore-frontend.git

cd mystore-frontend

Install Dependencies:

npm install


Start the Development Server:

npm start

Access the application:

Open http://localhost:3000 in your browser.
